# 05. 精神状態・自傷・希死念慮・飲酒

## 概要

- 配偶者は**婚姻前から2026年に至るまで一貫して希死念慮・鬱・パニック障害・アルコール多飲**を第三者に語り、精神科通院履歴を持つ
- 直接LINE（A）でも「死にたい」「消えたい」「鬱」の反復発言、パニック障害の自己診断、心療内科通院言及が多数
- **戦略的位置づけ**: ①離婚後の監護能力・生活能力への疑義、②離婚意思そのものの安定性への疑義（鬱期の情緒的判断で離婚を主張している可能性）、③婚姻継続困難の主因が配偶者側にあることの示唆
- **リスク**: 「配偶者を精神的に追い詰めたのはユーザー」と主張される反撃を想定し、婚姻前既往との切り分けを明確に

## 総括

- **武器としての活用**:
  - 配偶者の精神状態が婚姻中も継続的に不安定（希死念慮・アル依存・パニック）
  - ユーザーが**通院同行を継続的に申し出ていた**事実（姉ライン）は、婚姻継続への努力の重要証拠
  - 「離婚意思」が鬱期の判断であり冷却期間を要する主張の根拠
- **リスク**:
  - 「精神的に追い詰めたのはユーザー」と反撃される可能性 → **婚姻前既往（`04-dv-history.md`）と接続して切り分け**
  - ユーザー側の「鬱病でいーじゃん？！」等の発言は不利材料 → `08-user-risks.md` 参照
- **戦略**: 監護・親権議論（該当あれば）、および離婚意思の冷静さへの疑義提起で活用

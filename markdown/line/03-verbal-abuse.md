# 03. 暴言・モラハラ表現

## 概要

- **A（直接LINE）**では配偶者からユーザーへの直接的な「デブ」「キモい」「気持ち悪い」「クソ」「うざい」等の反復発言が多数記録されている
- **B（第三者経由）**では「犬」「犬コロ」「モラ夫」「旦那デスノート」「役に立たない夫」等の隠語・侮蔑表現が確認される
- ユーザーからの一部応戦的発言もあり（`08-user-risks.md` 参照）、双方向の可能性を認識する必要
- **戦略**: 配偶者が悪意をもってユーザーを侮辱していた事実は有責性議論の情状。ただしユーザー側の応答も相当数あるため、一方的モラハラでの慰謝料獲得は難しく、**「離婚を正当化しない程度の日常的口論」**として位置づけ、破綻否認の材料にする

## 総括

- **武器**: 「犬」「犬コロ」の反復侮蔑呼称、「旦那デスノート」愛用、「サイコパス」呼ばわり、「役に立たない夫」等の反復悪口は、**配偶者が家庭外でユーザーを継続的に貶めていた事実**として婚姻破綻の有責性を彼女側に寄せる材料
- **リスク**: A（直接LINE）ではユーザー側の応戦発言も相当数あるため、慰謝料獲得を目的とするより **「配偶者側にも同等以上の帰責事由あり」→有責回避** に用いる方が現実的
- **要注意**: 「気持ち悪い」の多くは配偶者自身の自嘲/一般話題。ユーザー宛と明確なもののみ最終的な提出用に絞り込む

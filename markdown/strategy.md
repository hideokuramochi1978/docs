# 戦略総括：離婚断念と有利離婚の両立

## 前提

- **優先順位1**: 配偶者に離婚を断念させる（婚姻継続）
- **優先順位2**: 離婚不可避の場合、有責配偶者化を回避し、財産分与で有利な条件を確保
- **ユーザー方針**:
  - 可能な限り弁護士をつけたくない（コスト・長期化・情報開示リスク回避）
  - 離婚不可避なら**財産分与なしの公正証書**での協議離婚が理想
  - 相手方が弁護士をつけたら「財産分与なし」は事実上不可能

## 証拠能力の区分（戦略設計の基礎）

### 証拠能力あり（原則、法廷でそのまま使える）
- **A: 直接LINE** = `hrk/トーク/[LINE] 百目とのトーク.txt`, `hrk/犬コロ/*` = ユーザーが当事者。**証拠能力あり**。ただし絵文字ロールプレイの解釈注意（README判別ルール参照）
- **その他**: ユーザー本人のメール、共有カレンダー、家計簿、通帳、診断書、公的書類、写真・動画（適法撮影）等
- **注**: 直接LINEでも、閲覧・保存の適法性（配偶者名義端末での勝手な取得ではない、共有クラウドから通常アクセスした等）は事前確認が必要

### 証拠能力に疑義あり（原則、法廷提出は避ける）
- **B: 第三者経由LINE** = `life/LINE/*`, `hrk/トーク/33XXX_*`, `hrk/トーク/YYYYMMDD/*` = 配偶者と第三者の会話。**配偶者のスマホから無断で抽出**（CLAUDE.md「戦略」節記載）
  - 不正アクセス禁止法・プライバシー侵害・私文書窃取のリスク
  - 法廷提出すれば逆に**ユーザー側の違法性**が争点化し、離婚事由に転化する可能性
- **戦略的位置づけ**:
  - **交渉段階での抑止・情報武装として内部利用**（相手方の主張を先読み、反論を用意）
  - **法廷提出は原則しない**。ただし相手方が同種のプライバシー侵害を主張してきた場合、対抗開示は選択肢
  - IMG_3917（性生活記録アプリ）も同区分

## 配偶者の想定行動パターンと対応

### パターン1: 「離婚したい」と単純に切り出される
**ユーザー対応方針**: 「好きだから離婚には同意しない」

**具体的手順**:
1. **感情的な受け止め**を優先し、その場で法的話をしない（「話し合いは大事にしたい、けれど今は感情の整理から」）
2. **書面化・録音を求められても応じない**（LINE等文字ベースの応答は簡潔に、意思表示のみ）
3. **別居を強行されないよう自宅から動かない**。生活費送金は継続（同居継続・扶助義務履行の実績づくり）
4. `07-cohabitation-life.md` の 2020-10-22「別居したいは無い」「ひでおは都合のいい男ではない」、姉ラインの「離婚するする言って10年以上」自認を**内部で確認**し、離婚意思が真意でない可能性を認識

**期待効果**:
- 配偶者が「単純に切り出す」段階では調停・弁護士まで進んでいない
- ユーザーが同意しなければ協議離婚は成立せず、配偶者は次の手段（弁護士）に進む必要が生じる
- そのハードルの高さで断念してくれる可能性を残す

配偶者は私のスマホを盗み見て**婚姻中（2025年末〜2026年初頭）の浮気未遂**を把握し、姉に共有している。私は配偶者のスマホから姉ライン等の非公式素材を抽出している。**双方が違法な入手経路で相手の情報を持っている**構造。ただし**私の事案は婚姻中**であり、相手方の攻撃材料としてのインパクトが大きい（1号不貞は未遂で否認できても5号情状として重い）。相互抑止のバランスは相手方に若干傾いている前提で運用する。

### 私の浮気未遂を配偶者が指摘してきた場合の対応

#### ケースA: 入手経路を伏せて「そういうことがあったんじゃないの？」と鎌をかけてくる
**リスク**: ここで否定すると、後日配偶者がLINEスクショ等を提示した場合、**「嘘をついた」＝信頼関係破綻の証拠**として5号事由に転用される。事実そのものより「隠蔽・欺瞞」が有責化の根拠になりやすい。

**取るべき対応**:
- **絶対に明確な否定はしない**。「嘘をついた」形成を回避することが最優先
- 曖昧な受け流し: 「何の話？」「どういう意味？」「昔のこと持ち出されても記憶が曖昧だな」
- 逆質問で情報源を探る: 「なんでそう思うの？」「何か見たの？」→ 配偶者が「スマホ見た」と自白すれば主導権が移る
- 議論を拒否: 「今その話をする気はない、話し合いたいなら別の機会に」
- **やってはいけない**: 「そんなことない」「してない」「潔白だ」等の明確な否認LINE・メール
- **記録**: 配偶者の指摘発言の日時・内容を自分側で残す（後日の反証用）
- **注**: この事案は**婚姻中**であり「昔・出会う前の話」として逃げることはできない。時期・態様・相手の話題を全て避け、事実の存否そのものに答えない方向で受け流す

#### ケースB: 入手経路を明かして直接「LINE見た。浮気してたよね、離婚したい／できるはず」と言ってくる
**この時点で配偶者は自白している**: プライバシー侵害・不正アクセス（配偶者間でも成立し得る）を自ら明かしたことになる。**主導権は私に移る**。

**取るべき対応**:
- 感情的に反応せず「それは私のスマホを勝手に見たということ？いつ？何を見た？」と**盗み見の事実を明確化させる**（記録・録音があれば残す。ただしLINE等文字ベースで自白させるのが最強）
- **事実部分については「未遂であり、判例上1号の不貞行為には該当しない」の一点に絞って反論**。時期・態様・相手の話には踏み込まない（**婚姻中の事案なので「婚姻前だから」の防御は使えない**。5号の情状としては攻撃材料になるので正面からの深追いは避け、法的評価だけで受ける）
- 離婚要求に対しては別軸で「離婚には同意しない」を維持
- **スマホ盗み見の違法性を強調**: 「あなたがしたことも問題で、法的には不正アクセスや私文書窃取に近い。あなたのLINEにも私の知らないことがあるかもしれないよね」で牽制（**姉ライン等の非公式素材の存在を仄めかす**足がかりにできる）

#### ケースC: 配偶者の弁護士から書面等で指摘される
- 相手方弁護士が浮気未遂を主張の柱にしてきた時点で、**入手経路の適法性が裁判所で問われる**構造になる
- こちらの対応:
  - 「その情報は配偶者が私のスマホを無断で見て取得したもの。証拠能力に疑義がある」と反論
  - 未遂であり判例上1号不貞行為に該当しないことを主張（婚姻中の事案なので時期による防御は使えない）
  - **同時に姉ライン等の非公式素材の存在も相手方に匂わせる**（相互違法収集のカード化）→ 相手方弁護士は「双方の違法性が争点化すると依頼人にも不利」と判断し、この論点を引く可能性

### 私が配偶者のLINEを盗み見たことを武器として使うタイミング

#### 弁護士がつく前（配偶者本人との直接交渉段階）
- **原則: 直接的には出さない**。「知っている」内容だけを踏まえた発言に留める
- 例: 「あなたも姉と離婚の話をずいぶんしてるみたいだね」等の**ぼかした示唆**は使える。ただし多用すると盗み見自白になる
- **最も安全な使い方**: 内容そのものを述べず、**「私も色々知っている」で止める**。相手が「何を知っているの？」と食いついてきても曖昧にし、心理的抑止だけ効かせる
- 具体的な引用・スクショ・日時特定は絶対に出さない（出した瞬間、私の側の違法性が確定する）

#### 弁護士がついた後
- **原則: 一切出さない**。相手方弁護士に対する示唆すら避ける
- 理由: 弁護士は違法収集情報の兆候を敏感に察知し、逆にこちら（私）を不正アクセス等で告訴・反訴する材料にする。相手方の攻撃強化を招く
- **例外**: 相手方が浮気未遂を書面主張してきた場合、**「相互違法収集」を主張する反論書面**の中で、姉ラインの存在を抽象的に言及することは選択肢（**弁護士に相談の上、私自身の弁護士経由でのみ**）。ただしこの段階では私にも弁護士が必要で、当初方針「弁護士なし」からは外れる

### 判断の指針

- **ケースA（鎌かけ）** で否定発言をしないこと ＝ 最重要。ここでの失敗が最大のリスク
- 私からの盗み見情報の使用は**「知っているぞという匂わせまで」**。具体引用に踏み込んだ瞬間、こちらが有責化のカードを渡す
- 相互違法性は**均衡している間だけカード**。片方が明示に踏み込めば均衡が崩れ、踏み込んだ側が不利になる

### パターン4: 配偶者が別居を開始／継続する

## 相手方が弁護士をつけるタイミングの読み

**弁護士着任の兆候**:
- 内容証明郵便が届く（協議離婚申入書 or 婚姻費用分担請求書）
- 配偶者から直接連絡が減り、代理人経由の連絡になる
- 家計・財産関係の資料開示を求められる
- 突然LINEでの応答が形式的になる

**タイミング予測**:
- 配偶者本人が「離婚したい」と切り出してユーザーが不同意 → 数週〜数ヶ月で弁護士検討
- 姉が煽動する形（姉ライン記録多数） → 弁護士紹介も姉経由の可能性大
- 別居中の生活費で揉めた場合、**婚姻費用分担調停**を先に申し立てられる可能性

**弁護士着任前にやるべきこと**:
1. **離婚届不受理申出**（最優先）
2. **家計・財産関係の記録整理**（通帳・給与明細・不動産・退職金見込み等を自分だけの管理下に）
3. **共同名義・共有クラウド等の見直し**（配偶者による今後の情報取得を制限）
4. **弁護士無料相談の下見**（複数弁護士に相談し、いざとなったら誰に依頼するか目星）
5. **本ドキュメント群の再整理と、法廷で使う証拠／使わない証拠の分別**

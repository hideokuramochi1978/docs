各トピックファイルの共通フォーマット：

```markdown
# <トピック名>

## 概要
（3行以内で、このトピックの証拠価値と全体傾向）

## 証拠一覧

### <見出し的な事象>
- **元ファイル**: [LINE] ますみとのトーク.txt
- **日付**: 2024-08-07 17:36（配偶者の発言日）
- **発言者**: はるか（百目）
- **引用**: 「鬱で死ぬかと思ったけど死ななかった！」
- **私にとってのリスク**: なし／〜な可能性
- **対策**: 〜
- **私にとっての武器**: 精神状態不安定の裏付け（監護権判断の材料）

```

「リスクと対策」「武器と活用方法」は該当がなければセクションごと省略。

## 抽出方針

### Phase 1: 素性・トーン把握
- 各ファイル冒頭200行 + 中盤 + 末尾を読み、話者と時期を確認
- はるかの呼称・自己認識（「百目」「デブ」など）を確認済み

### Phase 2: キーワード検索（各ファイルでgrep + 前後3行）

| トピック | キーワード |
|---|---|
| 不貞 | 浮気, 不倫, 好き, 会いたい, ホテル, 泊, 男, 元カレ, セックス, 抱, キス |
| 金銭 | ケチ, 金, 借金, 使い込, 給料, 通帳, 隠, カード, ブランド, 浪費 |
| 暴言・モラハラ | デブ, 死ね, 殺す, クソ, ムカ, キモ, 気持ち悪い, 嫌い, 別れ |
| DV既往 | 阿佐ヶ谷, 殴, 蹴, 怖, DV, 逃げ, 通報 |
| 精神状態 | 鬱, 病院, 心療, 眠れ, 死にたい, 限界, 薬, パニック |
| 飲酒 | 酒, 飲みすぎ, 二日酔, 泥酔, ワイン, ハイボール |
| 家事・生活 | 家事, 掃除, 洗濯, 料理, 主人, 夫, 旦那, 一緒に |
| 別居・離婚意思 | 別居, 離婚, 慰謝料, 弁護士, 出て行, 家を出 |
| ユーザーへの愚痴 | ケチ, うざ, ムカ, 帰ってこ, 話し合い |

### Phase 3: 分類と武器/リスク判定
- **武器**: はるか自身が第三者に語った浪費・不倫的言動・暴言・精神不安定・DV既往等
- **リスク**: はるかが第三者にユーザーの人格攻撃・悪評を流している事実（名誉毀損の余地 or 逆に離婚事由化）、および「ケチ」等ユーザー像がネガに描かれている箇所
- 各エントリを **元ファイル / 日付 / 発言者 / 短い逐語引用 / リスクと対策 / 武器と活用方法** の形で構造化

### Phase 4: 画像
- IMG_3916: 友人との中華会スクショ。婚姻中の飲食・外出頻度の記録
- IMG_3917: 2026-07-14「膣外射精」記録。**配偶者本人がスマホに保存していた記録として位置づけを保留** — 相手が誰か（ユーザー本人か第三者か）が不明なため、断定せず「性生活記録が2026-07-14に存在する事実」として記述し、ユーザー確認欄を設ける
